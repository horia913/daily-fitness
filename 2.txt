### 2. **Motion Design Language**

**Problem:** Static interface with no feedback or personality.

**Top Apps Solution:** Every interaction has a signature animation style.
```
ANIMATION DNA:

Page Transitions:
- Duration: 350ms
- Easing: Custom bezier (0.25, 0.46, 0.45, 0.94)
- Style: Slide with momentum, slight scale
- Never: Linear or instant cuts

Button Presses:
- Scale down: 0.94 (60ms, ease-out)
- Hold: Maintain scale
- Release: Spring back (180ms, spring damping 0.7)
- Haptic: Medium impact on press

Number Animations:
- Count up from 0 (or previous value)
- Duration: 800-1200ms based on magnitude
- Easing: Ease-out
- Add subtle "bounce" at end (spring)
- Decimal precision matches data type

Card Interactions:
- Hover/Press: Lift shadow, subtle scale (1.02)
- Dismiss: Slide + fade (300ms)
- Appear: Staggered fade-up (100ms delay between cards)

Loading States:
- Skeleton shimmer: 1400ms loop
- Shimmer angle: -20 degrees
- Colors: Background â†’ 10% lighter â†’ Background
- Never: Spinners for content loads
```

**Universal Rules:**
- NO instant state changes (always animate)
- Spring physics for natural feel (not ease-in-out)
- Haptic feedback for ALL taps (except scrolling)
- Exit animations should be 20% faster than entry
- Loading states should MORPH into content (not replace)

### 4. **Depth & Layering System**

**Problem:** Flat cards on flat background = boring, no hierarchy.

**Top Apps Solution:** Strategic elevation creates focus and premium feel.
```
ELEVATION LEVELS:

Level 0 (Background):
- Z-index: 0
- Shadow: None
- Color: Background color
- Use: App background, behind all content

Level 1 (Surface - Resting Cards):
- Z-index: 1
- Shadow: 0px 2px 8px rgba(0,0,0,0.08)
- Border: 0.5px rgba(0,0,0,0.04) (optional)
- Corner radius: 16pt
- Use: Default card state, list items

Level 2 (Raised - Important Cards):
- Z-index: 2
- Shadow: 0px 4px 16px rgba(0,0,0,0.12)
- Border: None
- Corner radius: 20pt
- Use: Hero cards, today's workout, streak display

Level 3 (Floating - Interactive):
- Z-index: 3
- Shadow: 0px 8px 24px rgba(0,0,0,0.16)
- Scale: 1.02
- Corner radius: 20pt
- Use: Pressed cards, dragging, FAB

Level 4 (Modal/Overlay):
- Z-index: 100
- Shadow: 0px 16px 48px rgba(0,0,0,0.24)
- Backdrop blur: 20pt
- Corner radius: 24pt (top corners only for bottom sheets)
- Use: Modals, bottom sheets, alerts

Gradient Overlays:
- Subtle gradient on all cards: 0 â†’ 2% white overlay
- Direction: 135deg (top-left to bottom-right)
- Creates depth and premium feel
- More pronounced in dark mode
```

**Universal Rules:**
- Cards should NEVER be flat against background (always shadow)
- Active/pressed states should visibly lift (increase shadow + scale)
- Overlapping content = higher z-index ALWAYS has more shadow
- Bottom sheets should blur background content
- Shadows should be colored (slight tint of primary color) not pure black

### 5. **Spacing & Layout Grid**

**Problem:** Inconsistent padding and margins throughout.

**Top Apps Solution:** Mathematical spacing system for rhythm.
```
8PT GRID SYSTEM:

Screen Margins:
- Horizontal: 20pt (iPhone), 24pt (iPad)
- Top safe area: 8pt below safe area
- Bottom safe area: 16pt above safe area + tab bar

Card Padding:
- External margin: 16pt between cards vertically
- Horizontal: 16pt from screen edge
- Internal padding: 20pt all sides
- Content to edge of card: 20pt minimum

Stack Spacing (Vertical):
- Micro (related items): 8pt
- Small (grouped content): 12pt
- Medium (sections within card): 20pt
- Large (separate sections): 32pt
- XLarge (major sections): 48pt

Horizontal Spacing:
- Icon to label: 12pt
- Button to button: 16pt
- Card to card (horizontal scroll): 16pt

Touch Targets:
- Minimum: 44x44pt (Apple HIG)
- Preferred: 48x48pt
- Comfortable: 56x56pt
- Use padding to expand hit area without expanding visual size

Grid Columns:
- 12 column grid for complex layouts
- 8pt gutter between columns
- Snap all elements to grid
```

**Apply Everywhere:**
- ALL measurements should be divisible by 4 (ideally by 8)
- Consistency > Perfection (16pt everywhere is better than 14pt/16pt/18pt mix)
- White space is a feature, not wasted space
- Dense information should still breathe (20pt minimum padding)

    const hour = new Date().getHours();

jsximport { useEffect, useRef } from 'react';
import { Animated, View, Dimensions } from 'react-native';

function FloatingParticles({ count = 15 }) {
  const particles = useRef(
    Array(count).fill(0).map(() => ({
      translateY: new Animated.Value(Dimensions.get('window').height),
      translateX: new Animated.Value(Math.random() * Dimensions.get('window').width),
      opacity: new Animated.Value(0),
    }))
  ).current;
  
  useEffect(() => {
    particles.forEach((particle, index) => {
      // Stagger start times
      setTimeout(() => {
        animateParticle(particle);
      }, index * 2000);
    });
  }, []);
  
  function animateParticle(particle) {
    particle.translateY.setValue(Dimensions.get('window').height);
    particle.opacity.setValue(0);
    
    Animated.parallel([
      Animated.timing(particle.translateY, {
        toValue: -50,
        duration: 30000,
        useNativeDriver: true,
      }),
      Animated.sequence([
        Animated.timing(particle.opacity, {
          toValue: 0.3,
          duration: 3000,
          useNativeDriver: true,
        }),
        Animated.timing(particle.opacity, {
          toValue: 0.3,
          duration: 24000,
          useNativeDriver: true,
        }),
        Animated.timing(particle.opacity, {
          toValue: 0,
          duration: 3000,
          useNativeDriver: true,
        }),
      ])
    ]).start(() => {
      // Loop
      animateParticle(particle);
    });
  }
  
  return (
    <View style={styles.particlesContainer} pointerEvents="none">
      {particles.map((particle, index) => (
        <Animated.View
          key={index}
          style={[
            styles.particle,
            {
              transform: [
                { translateY: particle.translateY },
                { translateX: particle.translateX },
              ],
              opacity: particle.opacity,
            }
          ]}
        />
      ))}
    </View>
  );
}

const styles = StyleSheet.create({
  particlesContainer: {

  particle: {

  };
  
  const rankChange = getRankChange();
  
  return (
    <Animated.View style={{ transform: [{ scale: scaleAnim }] }}>
      <Pressable
        onPressIn={handlePressIn}
        onPressOut={handlePressOut}

function getNextGoal(rank: number, workouts: number): string {
  if (rank > 100) {
    return `${Math.ceil((rank - 100) / 10)} more workouts to break top 100`;
  } else if (rank > 50) {
    return `${Math.ceil((rank - 50) / 5)} more workouts to break top 50`;
  } else if (rank > 25) {
    return `${Math.ceil((rank - 25) / 3)} more workouts to break top 25`;
  } else if (rank > 10) {
    return `${Math.ceil(rank - 10)} more workouts to break top 10`;
  }
  return "You're in the elite top 10! ğŸ†";
}

function formatDate(date: Date): string {
  const now = new Date();
  const diffMs = now.getTime() - date.getTime();
  const diffDays = Math.floor(diffMs / (1000 * 60 * 60 * 24));
  
  if (diffDays === 0) return 'today';
  if (diffDays === 1) return 'yesterday';
  if (diffDays < 7) return `${diffDays} days ago`;
  if (diffDays < 30) return `${Math.floor(diffDays / 7)} weeks ago`;
  if (diffDays < 365) return `${Math.floor(diffDays / 30)} months ago`;
  return date.toLocaleDateString();
}

typescript// utils/achievementTracker.ts
import { Achievement, AchievementUnlockEvent } from '../types/achievements';
import AsyncStorage from '@react-native-async-storage/async-storage';

class AchievementTracker {
  private achievements: Map<string, Achievement> = new Map();
  private listeners: Set<(event: AchievementUnlockEvent) => void> = new Set();
  
  async checkProgress(userId: string, metric: string, value: number) {
    // Check all relevant achievements
    for (const achievement of this.achievements.values()) {
      if (achievement.isUnlocked) continue;
      
      // Check if this achievement cares about this metric
      if (this.matchesMetric(achievement, metric)) {
        const previousProgress = achievement.progress;
        achievement.currentValue = value;
        achievement.progress = Math.min(
          (value / achievement.targetValue) * 100,
          100
        );
        
        // Check if just unlocked
        if (achievement.progress >= 100 && previousProgress < 100) {
          await this.unlockAchievement(userId, achievement);
        }
        
        // Save progress
        await this.saveAchievement(achievement);
      }
    }
  }
  
  private async unlockAchievement(userId: string, achievement: Achievement) {
    achievement.isUnlocked = true;
    achievement.unlockedAt = new Date();
    achievement.celebrationShown = false;
    
    // Save to storage
    await this.saveAchievement(achievement);
    
    // Track analytics
    this.trackUnlock(userId, achievement);
    
    // Notify listeners
    const event: AchievementUnlockEvent = {
      achievement,
      timestamp: new Date(),
      celebrationShown: false,
    };
    
    this.listeners.forEach(listener => listener(event));
  }
  
  onAchievementUnlock(callback: (event: AchievementUnlockEvent) => void) {
    this.listeners.add(callback);
    return () => this.listeners.delete(callback);
  }
  
  private matchesMetric(achievement: Achievement, metric: string): boolean {
    // Logic to match metrics to achievements
    const metricMap: Record<string, string[]> = {
      'workout_completed': ['volume_*', 'streak_*', 'consistency_*'],
      'pr_set': ['performance_*'],
      'leaderboard_rank': ['social_*'],
    };
    
    // Implementation details...
    return true;
  }
  
  private async saveAchievement(achievement: Achievement) {
    const key = `achievement_${achievement.id}`;
    await AsyncStorage.setItem(key, JSON.stringify(achievement));
  }
  
  private trackUnlock(userId: string, achievement: Achievement) {
    // Send to analytics service
    console.log('Achievement unlocked:', {
      userId,

      rarity: achievement.rarity,

    });

SplashScreen.preventAutoHideAsync();

export default function App() {

  const [appReady, setAppReady] = useState(false);
  
  useEffect(() => {
    async function prepare() {
      try {
        // Load performance settings
        const settings = await getOptimalSettings();
        setPerformanceSettings(settings);
        
        // Load fonts, data, etc.
        await Promise.all([
          // Add your loading logic here
        ]);

      } catch (error) {
        console.error('App preparation error:', error);

        await SplashScreen.hideAsync();

                case 'Workouts':
                  iconName = focused ? 'barbell' : 'barbell-outline';
                  break;
                case 'Nutrition':
                  iconName = focused ? 'nutrition' : 'nutrition-outline';
                  break;
                case 'Progress':
                  iconName = focused ? 'trending-up' : 'trending-up-outline';
                  break;
                case 'Profile':
                  iconName = focused ? 'person' : 'person-outline';
                  break;
              }
              
              return <Ionicons name={iconName} size={size} color={color} />;
            },
            tabBarActiveTintColor: '#FF6B35',
            tabBarInactiveTintColor: 'rgba(0, 0, 0, 0.5)',
            tabBarStyle: {
              backgroundColor: 'rgba(255, 255, 255, 0.95)',
              borderTopWidth: 0,
              elevation: 0,

              shadowOpacity: 0.1,
              shadowRadius: 8,
              height: 88,
              paddingBottom: 8,
              paddingTop: 8,
            },
            tabBarLabelStyle: {
              fontSize: 12,
              fontWeight: '500',
            },
            headerShown: false,
          })}
        >
          <Tab.Screen name="Dashboard" component={DashboardScreen} />
          <Tab.Screen name="Workouts" component={WorkoutsScreen} />
          <Tab.Screen name="Nutrition" component={NutritionScreen} />
          <Tab.Screen name="Progress" component={ProgressScreen} />
          <Tab.Screen name="Profile" component={ProfileScreen} />
        </Tab.Navigator>
      </NavigationContainer>
    </SafeAreaProvider>
  );
}
